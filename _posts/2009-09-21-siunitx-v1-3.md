I've just released version 1.3 of <a title="A comprehensive (sI) units package" href="http://tug.ctan.org/cgi-bin/ctanPackageInformation.py?id=siunitx">siunitx</a> to <a title="The Comprehensive TeX Archive Network" href="http://www.ctan.org/">CTAN</a>. Although the changes are not large in number, they are important. First, I've altered the default code for <code>\micro</code> (Î¼), as the previous version did not look great: an important change, hence the bump of version number. Second, the packaging has been altered to make life easier all round. You can now extract the package and typeset in one go from the .dtx file using LaTeX, or just extract it using TeX.  Overall, the effect is much cleaner than the old version. Of course, there are some bug fixes too.
